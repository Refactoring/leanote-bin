## 如何更新

请使用 `git pull` 更新, 或直接下载该仓库. 

为了安全起见, 请不要直接覆盖本地之前的二进制版, 请先备份数据与程序. 然后执行以下操作:

* 将该程序解压到另一目录
* 将之前的/conf/app.conf, /files, /public/upload, /mongodb_backup文件夹 拷贝到 新目录的相应位置
* 重新运行
